# 功能变更说明

## 一、变更概述

本次更新新增两项重要功能：
1. 学习资料管理功能
2. 积分奖励体系

---

## 二、新增功能详情

### 1. 学习资料管理功能

#### 1.1 功能说明
在考试之前，员工需要先完成相关学习资料的阅读和学习，管理员可以在管理端上传学习资料并将其与考试模板关联。

#### 1.2 管理端功能
- **学习资料管理页面**
  - 上传学习资料（支持PDF、Word、图片、视频等格式）
  - 编辑学习资料标题和描述
  - 删除学习资料
  - 查看学习资料列表
  - 将学习资料关联到考试模板

- **考试模板关联学习资料**
  - 在创建/编辑考试模板时，可选择关联的学习资料
  - 支持关联多个学习资料
  - 设置资料学习要求（是否必须完成学习才能参加考试）

#### 1.3 员工端功能
- **学习资料列表**
  - 查看需要学习的学习资料列表
  - **展示资料关联的考试信息**（重要：每个学习资料卡片上明确显示"为《XXX考试》准备"）
  - 按考试分类查看学习资料
  - 查看资料详情（支持PDF预览、图片查看、视频播放）
  - 标记资料为已学习
  - 查看学习进度
  - 显示该资料对应考试的学习要求（是否必须完成才能参加考试）

- **学习资料展示样式**
  - 资料卡片顶部显示关联考试名称（如："📚 为《电气安全操作考试》准备"）
  - 显示考试时间倒计时（如考试即将开始）
  - 显示学习状态：未开始/学习中/已完成
  - 显示学习进度条（如：3/5 已完成）

- **学习积分**
  - 完成单篇资料学习：+10积分
  - 完成全部关联资料学习：+20积分

### 2. 积分奖励体系

#### 2.1 功能说明
员工通过学习和考试获得积分，积分可用于激励员工积极参与学习和考试。

#### 2.2 积分获取规则
| 行为 | 积分奖励 |
|------|----------|
| 完成单篇学习资料 | +10积分 |
| 完成全部关联资料学习 | +20积分 |
| 考试及格 | +30积分 |
| 考试不及格 | +10积分（参与奖） |
| 考试满分 | +50积分 |
| 连续学习7天 | +50积分 |
| 连续学习30天 | +200积分 |

#### 2.3 管理端功能
- **积分管理页面**
  - 查看所有员工积分排名
  - 查看个人积分明细
  - 手动调整员工积分
  - 导出积分报表

- **积分规则配置**
  - 可自定义各项积分奖励数值
  - 可启用/禁用各项积分奖励

#### 2.4 员工端功能
- **我的积分**
  - 查看当前积分
  - 查看积分获取记录
  - 查看积分排名

## 四、数据模型变更

### 1. 新增学习资料表
```javascript
{
  id: Number,
  title: String,           // 资料标题
  description: String,     // 资料描述
  type: String,            // 资料类型（pdf/video/image/doc）
  url: String,             // 资料存储路径
  category: Number,        // 分类ID
  fileSize: Number,        // 文件大小
  duration: Number,        // 预计学习时长（分钟）
  createTime: String,      // 创建时间
  creator: String          // 创建人
}
```

### 2. 新增积分记录表
```javascript
{
  id: Number,
  userId: Number,          // 用户ID
  type: String,            // 积分类型（study/exam）
  action: String,          // 具体行为
  points: Number,          // 积分值
  description: String,     // 描述
  createTime: String       // 创建时间
}
```

### 3. 考试模板表变更
- 新增字段：`materials` - 关联的学习资料ID数组

### 4. 用户表变更
- 新增字段：`points` - 当前积分
- 新增字段：`totalPoints` - 历史累计积分
- 新增字段：`learningDays` - 连续学习天数
- 新增字段：`lastLearnDate` - 最后学习日期
